3D Recycle

What is it?

We have created a solution that creates value on plastic waste, making plastic waste a valuable material for common people.

Our solution to the problem is a 3D Printer that is able to melt down collected plastic materials and uses this to create a desired object. The solution creates a value on plastic waste, which will prevent plastics from entering the environment. With 3D printers you can print out 3-dimensional objects based on a working template. Manufacturers can provide you with a template where you can print a broken part of a machinery, you can make furniture, cutlery, tools even instruments to make a living off. This is how our innovation will create value on plastic waste. The goal is that people will start collecting plastic waste in their homes or in the environment, our solution will therefor prevent plastics from entering the environment and will reduce the material that is already out there.

How it works?

From plastic waste to plastic grace

The Print-Box is a 3d printer that is located on the streets. This printer is able to clean, meltdown and create 3d printing ink out of collected plastic materials.
Look up a product that you would like to make on the Print Box website or see available products on the machine itself. Get your code for the product.

A person empties his/her plastic materials into the machine. The material is washed and melted down into a material that is used as 3d ink. The person then enters their product code for the printing process to start.
